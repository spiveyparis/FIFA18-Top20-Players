# FIFA 18: Top 20 Player Analysis

This project analyzes the top 20 players in FIFA 18, showing their overall ratings, clubs, and nationalities. The interactive Tableau dashboard allows users to explore which clubs produce the most top-rated players and which players dominate the game.

## Files
- `fifa18_player_rankings.csv` — dataset of top 20 players
- `FIFA18_Top20.twbx` — Tableau workbook with interactive visuals

## Tools Used
- Oracle SQL (data extraction)
- Tableau (visualization)
